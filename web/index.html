Подключение к API

Нужны token и profile_id. Они сохранятся в браузере.

Где взять:
  1. Откройте school.mos.ru и войдите в аккаунт.
  2. DevTools → Application → Local Storage → https://school.mos.ru.
  3. Скопируйте ключи aupd_token и profile_id.
